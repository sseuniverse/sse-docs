## Usage

```vue [example.vue]
<script setup lang="ts">
const light = "https://avatars.githubusercontent.com/u/739984?v=4";
const dark = "https://avatars.githubusercontent.com/u/904724?v=4";
</script>

<template>
  <UColorModeAvatar :light="light" :dark="dark" />
</template>
```
